## 1. Purpose

This standard defines common terminology used throughout SEEH.

Consistent terminology prevents confusion across documents.

## 2. General Terms

| Term | Meaning |
|---|---|
| System | A group of blocks working together to perform a function. |
| Component | A single block or item within a system. |
| Module | A reusable system that can be copied into other builds. |
| Variant | A version of a design using different requirements or mods. |
| Interlock | A control rule that prevents unsafe actions. |
| Fail-safe | A design that defaults to a safer state when something fails. |
| Verification | Confirming that a system behaves as expected. |
| Maintenance Mode | A safe state for repair, inspection, or modification. |

## 3. Atmosphere Terms

| Term | Meaning |
|---|---|
| Airlock | A chamber used to transfer between pressurized and unpressurized areas. |
| Pressurize | Fill an area with breathable atmosphere. |
| Depressurize | Remove atmosphere from an area. |
| Vent | A block used to manage atmosphere. |
| Chamber | The enclosed space between airlock doors. |
| Interior Door | Door facing the protected interior. |
| Exterior Door | Door facing space, outside, or an unprotected area. |

## 4. Control Terms

| Term | Meaning |
|---|---|
| State | A defined operating condition of a system. |
| State Machine | A control method where the system moves between defined states. |
| Request | A player or system command asking the system to begin an action. |
| Override | A command that bypasses normal operation for special conditions. |
| Alarm | A signal that indicates a fault, emergency, or unsafe condition. |
| HMI | Human-machine interface, such as buttons, lights, or LCDs. |

## 5. Documentation Terms

| Term | Meaning |
|---|---|
| Standard | A rule or reference used across SEEH. |
| Guide | A step-by-step instructional document. |
| EDR | Engineering Decision Record. |
| Released | Document is stable and ready for general use. |
| Draft | Document is incomplete or not fully verified. |
| Experimental | Design or procedure requires further testing. |
